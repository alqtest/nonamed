# 2026-02-19 - Cloudflare Tunnel

## Pasos para crear un túnel de Cloudflare

1. **Instalar cloudflared:**
   - Linux ARM64: `wget https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-arm64.deb && sudo dpkg -i cloudflared-linux-arm64.deb`

2. **Login:**
   `cloudflared tunnel login`

3. **Crear túnel:**
   `cloudflared tunnel create <NOMBRE>`

4. **Configurar:**
   Crear `~/.cloudflared/config.yml`:
   ```yaml
   tunnel: <NOMBRE>
   credentials-file: /home/ubuntu/.cloudflared/<NOMBRE>.json
   
   ingress:
     - hostname: tudominio.com
       service: http://localhost:PUERTO
     - service: http_status:404
   ```

5. **Levantar:**
   `cloudflared tunnel run <NOMBRE>`
   O como servicio: `sudo cloudflared service install <NOMBRE>`

## Notas
- Necesitas un dominio en Cloudflare
- Abrir puerto 443 saliente en firewall
- Mantener el credentials file seguro
